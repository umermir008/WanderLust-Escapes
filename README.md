# Wanderlust Escapes

A luxury travel website built with HTML, CSS, and JavaScript.

## Running the Website Locally

### Prerequisites
- Node.js installed on your system

### Quick Start
1. Open a terminal/command prompt in this project directory
2. Run one of the following commands:

```bash
# Run on default port 3000
npm start

# Or run on specific ports:
npm run dev-3001  # Port 3001
npm run dev-8080  # Port 8080
npm run dev-5000  # Port 5000
```

### Manual Port Selection
You can also run the server on any port by using:
```bash
node server.js [port_number]
```

For example:
```bash
node server.js 4000  # Runs on port 4000
```

### Accessing the Website
Once the server is running, open your browser and navigate to:
- http://localhost:3000 (or whichever port you selected)
- http://127.0.0.1:3000 (alternative address for the same server)

### Stopping the Server
Press `Ctrl + C` in the terminal to stop the server.
